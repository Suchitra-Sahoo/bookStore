## üöÄGetting Started

### Installation

1. **Clone this repository:**

    ```bash
    git clone https://github.com/Suchitra-Sahoo/bookStore.git
    ```

2. **Install Frontend dependencies:**

    ```bash
    cd Frontend
    npm install
    ```

3. **Install Backend dependencies:**

    ```bash
    cd Backend
    npm install
    ```

4. **Running the Frontend:**

     ```bash
    cd Frontend
    npm run dev
    ```
   
5. **Running the Backend:**

    ```bash
    cd Backend
    npm start
    ```

After completing these steps, you should have the bookStore website up and running on your local machine. üéâ
